# CZPPlayer
## 基于Qt实现的简易音乐播放器(使用Phonon框架)
### 基本功能
* 上一首
* 下一首
* 暂停&播放
* 列表隐藏和显示
* 主界面&mini界面
* 添加播放列表
* 从本地批量导入歌曲
* QSS界面美化
* SQLite缓存本地数据
